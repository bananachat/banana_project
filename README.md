## 1. 프로젝트 소개
바나나톡은 Java 소켓통신과 Oracle DB를 이용한 멀티 채팅 프로그램입니다.
<br><br>

## 2. 설치 방법
1. 자바17, 오라클 설치
2. 클론 후 db테이블설치
3. db, 서버주소 확인
4. 서버 실행 후 클라이언트 실행
<br><br>
